# 🎮 Online Game Store with Chat System

A full-stack web application developed using PHP and MySQL that allows users to browse, download, and interact with games. The platform also enables developers to upload games and communicate with users through a real-time chat system, along with an admin dashboard for managing the entire system.

---

## 🚀 Features

* 🕹 Browse and download games
* 👤 User registration and login system
* 💬 Real-time chat between users and developers
* 📤 Game upload and management for developers
* 🛠 Admin dashboard for full system control
* ⭐ Game ratings and reviews
* 📂 Category and subcategory management

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP (Apache)

## ⚙️ Installation & Setup

1. Install XAMPP (Apache & MySQL)

2. Clone this repository:
   git clone https://github.com/your-username/online-gamestore.git

3. Move the project folder to:
   xampp/htdocs/

4. Open phpMyAdmin:
   http://localhost/phpmyadmin

5. Create a database:
   db_miniproject

6. Import the SQL file into the database

7. Start Apache and MySQL in XAMPP

8. Open in browser:
   http://localhost/MinProject/Project/
